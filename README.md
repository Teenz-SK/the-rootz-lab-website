# 🌿 The Rootz Lab – Premium Hair Restoration Website

## 🚀 Live Website

🔗 https://therootzlab.co.in/

---

## 💎 Project Overview

A modern, premium UI-based business website **developed for a real client (The Rootz Lab)**, focusing on non-surgical hair restoration services.

Designed with a **high-end SaaS-style UI**, smooth animations, and a conversion-focused layout to maximize customer engagement and lead generation.

---

## 🛠 Tech Stack

* HTML5
* Tailwind CSS (CDN)
* Vanilla JavaScript

---

## ✨ Key Features

* 🎨 Premium UI/UX design (glassmorphism, gradients, modern layout)
* 📱 Fully responsive (mobile-first + desktop optimized)
* ⚡ Smooth animations using AOS (Animate On Scroll)
* 🎯 Conversion-focused layout for client lead generation
* 📞 Direct Call & WhatsApp Integration
* 📩 Lead form connected to WhatsApp (real-time inquiry system)
* 🖼 Image sliders, testimonials, and service showcase sections

---

## 📚 Libraries Used
